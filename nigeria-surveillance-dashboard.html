🔍 Source Audit

✅ Live Feeds Active

  • APM Terminals Apapa — Official gate camera images at cms-cd.apmterminals.com (refreshed ~60s). These are the only real, verified Nigeria-specific live CCTV images.
  • Google Maps Traffic tiles — Embedded maps zoomed to major roads with real-time traffic layer. Updated by Google continuously.
  • NTA (Nigerian Television Authority) — YouTube live embed where available.
  • Channels TV — YouTube live embed for Nigeria news context.

⚠️ Why Some Feeds Don't Load

  • Pictimo / insecam — Returns CORS-blocked images. Visible in browser but canvas motion analysis is blocked. Old thumbnails, not live.
  • TrafficEye.com.ng — Verified as a landing page only; no embeddable stream endpoints found.
  • naijacams.com — DNS resolution failed on verification (April 2026).
  • WeatherBug Abuja/Lagos — Reports "no traffic cams in region."
  • LASTMA cameras — Not publicly embeddable; used internally only.

🚀 To Get More Real Cameras

Set up a CORS proxy (e.g., Cloudflare Worker) or contact these sources directly:

  • trafficeye.com.ng — Nigerian traffic portal
  • APM Terminals Apapa gate cams
  • WorldCam Nigeria
  • RTSMG (Road Transport Safety)

📡 Motion Analysis

For <img> feeds with CORS headers, pixel-diff runs every 5s via hidden canvas. Threshold: >3% changed pixels = traffic motion. Night (20:00–05:00 WAT) + spike = security alert. No motion 30s+ = signal lost.
